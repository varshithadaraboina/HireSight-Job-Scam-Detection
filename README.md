HireSight – AI-Based Job Scam Detection Platform

Introduction
HireSight is a web-based platform designed to identify fraudulent job postings. With the increasing number of online job scams, many users fall victim to fake offers that demand money or personal information. This system helps users detect such scams by analyzing job descriptions and providing warnings.

Objectives
- Detect fake job postings  
- Protect job seekers from fraud  
- Provide scam risk analysis  
- Verify company authenticity  
- Allow users to report suspicious jobs  

Stakeholders
- Job Seekers  
- Students and Fresh Graduates  
- Companies  
- Recruitment Platforms  
- Cybercrime Authorities

Features
- Job Scam Detection  
- Company Verification  
- Scam Reporting System  
- Risk Score Generation  

Workflow
1. User enters job description  
2. Frontend sends data to backend  
3. Backend analyzes text using rule-based logic  
4. Scam risk score is generated  
5. Result is displayed to the user  

Example Output

Risk Level: High
